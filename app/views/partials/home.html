NTU STARS Planner Assistant

A simple tool that helps you find your plan faster


Plan Finder

Step 1. Drag the Plan Finder bookmarklet above to your browser's bookmark bar.

Step 2. In STARS Planner, select all courses that you want to have.

Step 3. Go back to STARS Planner's main page and click this bookmarklet.

Step 4. A new window or new tab will be opened to find possible index combinations.

What it DOES

  • It quickly checks if it is possible for you to take a set of courses and return to you the set of indexes if it is possible. The check is done based on course's time slot and exam date.

  • It can show hidden indexes that aren't shown up in STARS Planner and you might be able to appeal for.

What it DOES NOT

  • It doesn't register the course for you. You still have to go to STARS Planner and register your courses in STARS.

  • It doesn't guarantee you can take the course if you don't meet the course's pre-requisition. We don't have that data of you.

  • It doesn't spam NTU server to get you more chances to get some course, because that's strictly prohibited.

  • It doesn't send a single HTTP request to NTU server. All requests are sent to our own server developed and maintained by this application's developer.

So is this allowed?

For short, we don't guarantee anything, yet. However, the app is being sent for NTU's approval. The result is expected to be bright, because

  • It isn't unfair to use this tool, for anyone can use it.

  • The app doesn't add any additional load to NTU server, not a single request is sent to NTU's server. This can be monitored using browser dev tool by clicking F12 -> Network inside your browser.

  • In consequence, it even decrease the load for NTU server because most of the try & false work is done in this application's environment rather than STARS Planner.

You can try using this application at your own risk, we will soon update the approval status as soon as we hear from school.
